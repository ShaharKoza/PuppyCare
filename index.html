Overview

PuppyCare is a full-stack IoT product combining a Raspberry Pi sensor station with a native iOS app. It gives dog owners real-time visibility into their kennel environment: temperature, humidity, motion, light, sound, and live camera snapshots — all surfaced through an intelligent, design-forward mobile interface.


The system uses Firebase Realtime Database as the communication backbone between the Pi and the iPhone, and Firebase Cloud Messaging to push critical alerts even when the app is in the background.


The problem it solves: Dog owners who use kennels have no way to know if their dog is too hot, too cold, anxious, or in distress. PuppyCare closes this gap with live environmental monitoring, behavioral pattern analysis, and an intelligent feeding and care management system.

Features

📡

Live Sensor Dashboard

Real-time temperature, humidity, motion, bark detection, light state, and kennel presence timer — all updating live via Firebase.

🚨

Smart Alerts System

Automatic alerts for out-of-range temperature, bark escalation, motion, and light changes. Full alerts history with filter tabs, search, and behavioral analytics.

📷

Camera Integration

On-demand snapshot capture: the app writes a request to Firebase; the Pi takes a photo and uploads the URL back for instant display.

🍖

Feeding Management

Daily schedule with meals, walks, and play sessions. Per-meal gram tracking, calorie calculation, and next-event badge per routine row.

🤖

AI Food Assistant

Type any food name and get a safety assessment (safe / caution / dangerous) with explanation and tips — powered by a rule-based engine with pluggable LLM protocol.

🐕

Dog Profile

Breed, sex, age, weight, custom alert thresholds, Israel-specific vaccine reminders, and profile photo from the camera roll.

🔔

Push Notifications

Firebase Cloud Functions trigger FCM push notifications for warning, stress, and emergency alert levels — even when the app is in the background.

📊

Behavioral Analytics

Peak bark hour, most active period, temperature range summary, and a 12-hour bark frequency bar chart — surfaced from local alert history.

✨

Guided Onboarding

First-launch setup flow collecting dog name, breed, sex, age, and weight before showing the main interface — zero friction for new users.

Raspberry Pi reads sensors every few seconds and writes structured JSON to Firebase paths.

2

FirebaseService holds active listeners on each path and updates @Published state on the main actor.

AlertManager applies cooldown and threshold logic, appending AlertRecord entries to persistent store.

Cloud Function triggers on kennel/alert writes and sends FCM push notifications to the registered device token.
