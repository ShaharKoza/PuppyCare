Overview

What is PuppyCare?

PuppyCare is a full-stack IoT product that closes the information gap between dog owners and their kennels. By pairing a Raspberry Pi sensor station with a native SwiftUI iOS application, PuppyCare delivers continuous, real-time visibility into every aspect of a puppy's environment — temperature, humidity, motion, sound levels, light conditions, and live camera snapshots.

The core problem PuppyCare solves is ambient uncertainty: when a dog is kenneled, its owner has no reliable way to know whether conditions are safe or distressing. Extreme temperatures, sustained barking, or irregular motion patterns can go unnoticed for hours. PuppyCare eliminates this gap with a real-time data pipeline built on Firebase Realtime Database — sensor data written by the Pi propagates to the iOS app in under a second, and Firebase Cloud Functions dispatch push notifications for any critical event.

Beyond monitoring, PuppyCare is a complete care management platform. An integrated feeding scheduler tracks meals, walks, and play sessions with per-meal calorie accounting. An AI food safety assistant evaluates any ingredient for canine safety in real time. A detailed dog profile stores breed-specific thresholds and generates personalized vaccine reminders — making PuppyCare the single app every puppy owner needs.

Features

Everything your puppy needs.

From live environmental monitoring to a built-in AI food assistant — PuppyCare covers the full spectrum of modern kennel care.

📡

Live Sensor Monitoring

Real-time tiles for temperature, humidity, motion, bark count, light state, and kennel presence timer — all updating live via Firebase listeners.

🚨

Smart Alerts

Threshold-based alerts for extreme temperatures and sustained barking. Firebase Cloud Functions dispatch FCM push notifications to your device instantly.

📷

Camera Updates

Tap to request a live kennel snapshot. The Pi captures and uploads the image to Firebase; the app displays it within seconds with a timestamp.

🍖

Feeding Management

Daily meal schedule with gram tracking and automatic calorie calculation. Add meals, walks, and play sessions with next-event countdowns on every row.

📅

Routine Tracking

Structured daily routines with local notifications for upcoming meals and walks. Never miss a scheduled care event — even when the app is in the background.

🐾

Dog Profile & Reminders

Breed, sex, age, weight, and custom alert thresholds stored locally. Auto-generated vaccine reminders and an AI food safety assistant built right in.

Technical Details

Under the hood.

A purpose-built stack that prioritises real-time performance, native iOS quality, and serverless scalability.

🏗️ App Architecture

  • UI Layer SwiftUI, Combine, @MainActor — all views reactive to @Published sensor state
  • Core Services FirebaseService · AlertManager · ProfileStore · FoodAssistantService · ReminderManager
  • Data Flow Pi → Firebase RTDB → iOS listeners → UI re-render under 1 second end-to-end
  • Design System Custom AppTheme with semantic color tokens, corner radii, spacing, and typography constants

🔧 Raspberry Pi + Sensors

  • Platform Raspberry Pi running a Python sensor script with Firebase Admin SDK
  • Temp & Humidity DHT22 polled every 5 s — written to kennel/dht
  • Motion PIR sensor with last-motion timestamp and seconds-since counter
  • Sound & Camera Sound module with 5-second burst counting; camera responds to capture_request writes

☁️ Firebase + Notifications

  • Realtime DB Structured under kennel/ — dht · pir · sound · light · alert · camera paths
  • Cloud Functions Node.js 18, Functions v2 — triggers on kennel/alert where level ≠ "normal"
  • Push Notifications FCM + APNs — device token stored at kennel/fcm_token by the iOS app
  • Security GoogleService-Info.plist gitignored; Firebase service account on Pi only

User Guide

Getting started.

Everything you need to set up, configure, and use PuppyCare from day one.

1

Onboarding

On first launch, a guided flow collects your dog's name, breed, sex, age, and weight before showing the main interface.

2

Profile Setup

Set breed-specific alert thresholds for temperature and sound. Add a profile photo. View auto-generated vaccine reminders based on age.

3

Dashboard Usage

The Dashboard shows all live sensor tiles. Tap the camera card to request an on-demand snapshot from the kennel Pi.

4

Feeding & Routine

Add meals, walks, and play sessions. Set gram amounts for automatic calorie totals. Routine rows show a next-event countdown.

5

Alerts & Reminders

Browse the date-grouped alert history. Use filter chips to narrow by event type. Analytics show daily bark patterns and activity summaries.

6

Camera & Notifications

Grant push notification permission on first launch. Critical and warning-level events push via FCM even when the app is fully closed.
